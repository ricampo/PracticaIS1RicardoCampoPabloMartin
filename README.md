# PracticaIS1RicardoCampoPabloMartin

Este proyecto consiste en una calculadora modular básica desarrollada en C++, creada como práctica de manejo de Git y GitHub en equipo.
El código está dividido en tres módulos principales para fomentar la colaboración entre los integrantes:

Módulo INOUT → Gestión de entrada/salida por consola.

Módulo OPERATIONS → Implementación de las operaciones matemáticas.

Módulo MAIN → Lógica principal del programa.

La aplicación permite realizar operaciones aritméticas simples y muestra los resultados.
